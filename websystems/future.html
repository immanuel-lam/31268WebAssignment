Future Aspirations

I have a pretty clear idea of where I want to end up, which is offensive cybersecurity. Red teaming and pen testing specifically, basically where you approach things the way an attacker would and try to find the holes before someone actually exploits them.

Career Goals

I want to work as a penetration tester or red team operator, ideally at a dedicated security firm or on an internal security team. What I like about it is how much technical depth it actually requires. You need to understand why something is exploitable, not just that it is.

Some areas I want to build skills in:

  • Web application penetration testing
  • Network penetration testing
  • Exploit development and vulnerability research
  • Reverse engineering
  • Active Directory and Windows environments

Certifications I Want to Pursue

Certs aren't everything but they give a structured path. The ones I'm aiming for:

Certification Why
OSCP (Offensive Security Certified Professional) Industry standard for penetration testing. Hands-on exam, no multiple choice.
eJPT (eLearnSecurity Junior Penetration Tester) Good starting point before OSCP, practical and affordable
CompTIA Security+ Broad foundation, valued by employers

Technologies to Learn

A few areas I want to go deeper in over the next few years:

  • C and low-level programming (understanding how memory works)
  • Assembly language basics (useful for reverse engineering)
  • More advanced Linux administration and scripting
  • Cloud security (AWS, Azure environments)

Further Study

I plan to finish my Bachelor of Cybersecurity at UTS and see where things go from there. Postgrad is possible down the line but that's pretty far off right now.

Home Lab Going Forward

I'll keep building out my home lab. Eventually I want a dedicated segment with intentionally vulnerable VMs and CTF-style boxes to practice on, because there's only so much you can learn from just reading about it.
