Future

I want to work in red team / offensive security. Still a first year student so that's a few years off, but it's the direction I'm pointed in.

Why red team?

I find defence kind of boring, honestly. You're mostly waiting for something to go wrong. Red team you're the one making things go wrong, which requires actually understanding how systems work at a deeper level.

Running a home lab has helped with this more than I expected. When you set up a reverse proxy and expose it to the internet, you start thinking about what an attacker would do with a misconfiguration pretty quickly.

Areas I'm interested in

Penetration testing
The job I actually want. Scoped assessments where you find vulnerabilities and write them up for whoever hired you. The OSCP is the cert I'm working toward eventually.
CTFs (Capture the Flag)
I spend more time on Hack The Box than I probably should. It covers web exploitation, binary exploitation, crypto, forensics, and rev. The machines range from easy to deeply humbling. Good practice.
Exploit development
Buffer overflows, use-after-free, format strings. This is the part that requires actual C and assembly knowledge, which I'm still building. It's slow going but interesting.
Active Directory
Most corporate networks run on AD, so most red team work involves it at some point. Kerberoasting, Pass-the-Hash, lateral movement — I've been reading about these and setting up a lab environment to practice.
Web application security
SQL injection, XSS, IDOR, SSRF — the OWASP Top 10 stuff. Having built web apps myself makes it easier to understand where these bugs come from.
OSINT
Recon before you touch anything. Useful for understanding how much of an organisation's attack surface is just sitting there publicly. Also genuinely interesting as a skill on its own.

What I'm doing now

  • First year at UTS, picking security-relevant electives where I can
  • Hack The Box and TryHackMe on weekends
  • Reading The Hacker Playbook and Georgia Weidman's Penetration Testing
  • Building a separate attack lab VM alongside the home lab
  • CompTIA Security+ at some point, then OSCP when I'm ready
