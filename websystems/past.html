How It Started

Like a lot of people my age, computers started with gaming. At some point I stopped just playing and started actually wanting to know how everything worked. I built my own Windows PC and started digging into it properly, things like overclocking and triple-booting with OpenCore. From there it kind of snowballed.

Networking and Home Lab

A couple of years ago I started building a home lab. It started small but grew into a proper setup: a QNAP NAS, a Mac Mini running as an always-on server, KVM virtualisation, and a UniFi network with VLANs and multiple access points. I run Home Assistant for home automation and have integrated a Sigenergy solar and battery system through a local Modbus connection.

I also self-host stuff like Vaultwarden (a password manager with WebAuthn and YubiKey support) and run local LLMs using LM Studio and Ollama.
