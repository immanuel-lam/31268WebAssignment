How It Started

Started with gaming. At some point tweaking the PC got more interesting than actually playing - overclocking, getting OpenCore working for a triple boot setup, that sort of thing. Once I started pulling apart how the networking worked I didn't really stop.

Home Lab

Built up a home lab over a few years. Currently running a QNAP NAS, a Mac Mini as an always-on server, KVM for virtualisation, and a UniFi network with VLANs across a few access points. Also have Home Assistant set up and integrated a Sigenergy solar and battery system through a local Modbus connection which took a while to get working.

Self-host Vaultwarden for passwords (with YubiKey and WebAuthn), and run local LLMs through LM Studio and Ollama when I need them.
